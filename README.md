# Iran War: Strait of Hormuz Impact

Interactive infographic with live RSS news feed.

## Files
- `index.html` — Main infographic (open in any browser)
- `thumbnail.png` — 1200×630 OG share image

## Features
- Real map (OpenStreetMap) with strike locations, naval assets, pipelines
- Live RSS headlines from BBC, Reuters, CNN, Al Jazeera, Sky News, AP
  - Auto-refreshes every 5 minutes
  - Filters for Iran/Hormuz keywords
- EIA data: oil flow, bypass capacity, destination markets
- Escalation timeline

## Hosting
Drag `index.html` to netlify.com/drop for instant live URL.

## Sources
- U.S. EIA (eia.gov) — oil flow data
- Vortexa — tanker tracking data
- AP, Reuters, Al Jazeera, The Guardian — breaking news
- Wikipedia — 2026 Iran–US crisis timeline
